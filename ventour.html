what.

Mobile App Development.

when.

April 2020.

why.

Learn a build process with a mobile app framework, and create a rapid prototype.

background.

This was a school project that spanned across two classes; Mobile App Development and Interface Design. The process started in Interface Design where I was tasked with creating a concept for an app of my creative choice. After finalizing a concept, developing a plan and creating rough wire-frames I then fleshed out a complete design in XD along with user stories.

The continuation of this project stretched into a Mobile App Development class where I was tasked to completely build the front end of my designs with all views/pages, and front end functionality. The idea was to have a working prototype that could be scaled into a full stack application.

tasks completed.

  1. Created wire-frames, fleshed out UI designs and user stories.
  2. Developed a responsive and interactive UI in the mobile app framework Nativescript using its components.
  3. Developed front end functionality such as data binding and conditional logic for scalability.
