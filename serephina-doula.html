serephina doula.

what.

Web Design and Development.

when.

April 2021.

why.

To reach customer base.

background.

Serephina is a full spectrum doula based in North Western Ontario with a wide array of family support services that range from remote and in-person postpartum programs to reiki sessions. They were just starting out and had some branding assets in place, as well as a vision but weren't sure where to go from there. So I began an intake process and settled on the deliverables of a polished set of brand guidelines, a website design, and finally a web development that could be easily maintained by themselves.

tasks completed.

  1. Polish current brand assets for use in a set of brand guidelines that will define all aspects of content moving forward.
  2. Create an information architecture that directly reflects client/user needs.
  3. Design a full prototype of the website using Adobe XD for review and correction.
  4. Develop a custom WordPress application using a child theme while taking into account site performance and ease of editing for the client. Serephina Doula wanted to be able to edit almost all aspects of the site themselves.
  5. Implementation of Calendly booking for each service on the website.
  6. Website integration with all social media platforms and email.
